                    //Check to see if word is palindrome
                    function palindromeCheck(inputVal) {
                      //Matches anything that is A-Z, a-z, and 0-9, everything else is removed. Also searches globally.
                      const sub = /[^A-Za-z0-9]/g;
                    
                      //Changes all letters to lower case
                      inputVal = inputVal.toLowerCase();
                    
                      //Uses the regEx expression to remove anything that doesnt match.
                      inputVal = inputVal.replace(sub, '');
                    
                      //splits the formmating string into an array of characters, reverses the array,
                      //and joins the reverse array to form the reversed string
                      const reverseFormattedString = inputVal.split('').reverse('').join('');
                    
                      //object to store the return message and reverse string.
                      const resultObj = {};

                        resultObj.msg = `Great! You're value is a palindrome! ðŸŽ‰ðŸŽ‰ðŸŽ‰ðŸŽ‰`;

                      }
                    
                      resultObj.returnString = reverseFormattedString;
                    
                      return resultObj;
                    }
                    
                    //Display result to user
                    function displayResults(obj) {

getValues Function

To use the getValues function, we get the input value that the user types in, then we pass it along to the isPalindromCheck function. Once that function returns a result, we pass it to the displayResults function.

palindromeCheck Function

In the palindromeCheck function, we first declare a variable named sub that holds our RegEx expression. Then we take the input passed in, and change each letter to lowercase. Next, we run the replace method on the string using our RegEx variable. Anything that doesnâ€™t match, we replace with an empty string. With this clean string, we split each letter and put it into an array, reverse the array, and join each letter back into a string for comparison. The message in the object will change considering if the string is a palindrome or not. Once weâ€™ve stored the message, we add the reverse string and return the object to the caller.

displayResults

The displayResults function will check to see if the message contains a part of the success message. If it does, it will set the alert element to the success color before displaying the message. Otherwise we choose the color of danger. With the color set, it will display the right message along with the string used to make that determination.
